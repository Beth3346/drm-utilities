Time Utilities

A series of methods that make it easier to work with dates and times.

Now

Returns a new date object with the exact date and time.

var now = elrTime.now;

Today

Returns an object with the exact date and time

var today = elrTime.today;
today.month // 4
today.day // 0
today.date // 31
today.year // 2015
today.hour // 19
today.minute // 47
today.second // 43

Days Per Week

Number of days in a week.

var daysPerWeek = elrTime.daysPerWeek; // 7

Unit Tokens

Unit tokens for use in date formatting.

var tokens = elrTime.unitTokens;
tokens.ms // millisecond
tokens.s // second
tokens.m // minute
tokens.h // hour
tokens.d // day
tokens.D // date
tokens.w // week
tokens.M // month
tokens.Q // quarter
tokens.y // year
tokens.DDD // dayOfYear
tokens.a // ampm

Months

Array of long month names

    var months = elrTime.months;

    Short Months

    Array of short month names

      var months = elrTime.months;

      Days

      Array of long day names

        var days = elrTime.days;

        Short Days

        Array of short day names

          var shortDays = elrTime.shortDays;

          Min Days

          Array of minified day names

            var minDays = elrTime.minDays;

            Is Leap Year

            Returns true if a year is a leap year.

            • 1900
            • 2000
            • 2004
            • 2006
            • 2007
            • 2015
            • 2016
            elrTime.isLeapYear(1900); // false
elrTime.isLeapYear(2000); // true

            Days in Year

            Returns the number of days in a given year.

            • 1900
            • 2000
            • 2004
            • 2006
            • 2007
            • 2015
            • 2016
            elrTime.getDaysInYear(1900); // 365
elrTime.getDaysInYear(2000); // 366

            Days in Month

            Returns the number of days in a given month.

            • February, 1900
            • February, 2000
            • March, 2004
            • June, 2006
            • August, 2007
            • December, 2015
            elrTime.getDaysInMonth(month, year); // January is 0
elrTime.getDaysInMonth(1, 1900); // 28
elrTime.getDaysInMonth(1, 2000); // 29

            Day of Week

            Returns the index of the day of the week

            elrTime.getDayOfWeek(elrTime.today.month, elrTime.today.date, elrTime.today.year);

            First Day of Month

            Returns the index of the first day of the month

            • May, 2016
            • June, 2016
            • March, 2016
            • June, 2016
            • August, 2016
            • December, 2016
            elrTime.getFirstDayOfMonth(month, year);
elrTime.getFirstDayOfMonth(5, 2015); // 1
elrTime.getFirstDayOfMonth(4, 2015); // 5

            Weeks in Month

            Returns the number of weeks in a given month

            • May,2015
            • June,2015
            elrTime.getWeeksInMonth(month, year);
elrTime.getWeeksInMonth(5, 2015); // 6
elrTime.getWeeksInMonth(4, 2015); // 5
